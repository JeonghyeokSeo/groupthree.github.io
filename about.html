Our Approrach

Our team develop every project using two lifecycles, one being the project management lifecycle and the other being the system development lifecycle.

Project Management Lifecycle

This lifecycle contains four stages, which ensures a project is managed properly. The project manager is responsible for the proper implementation of this lifecycle and it involves consultation of the client.

Initiation:

During this stage the the business need, feasibility, initial requirments and priorites of a project are determined. This stage produces a project charter, which contains the projects objectives, how the project will be executed, and the project stakeholders.

Planning:

During this stage the project manager plans the project. This entails determing the scope, schedule, and budget of the project. This stage will produce the project plan.
Execution and Control:

This stage is when T3 GIS Consluting's technical team executes the project. The project manager will monitor the project progress and produce status reports for clients. Also clients can schedule meetings where the project manager provides updates on how the project is progressing.

Closeout:

The last stage of the lifecycle is where the T3 GIS Consulting Teams reflects on lessions learned from the project. This stage involves the project manager conducting a project review meeting, where all teams members and the client dicuss the implementation and success of the project.

System Development Lifecycle

This lifecycle is mainly conducted by the T3 GIS Consulting's technical team and does not really invovle the client. There is five steps to this lifecycle.

Initiation:

This stage is similar to initiation stage of the project management lifecycle.

Design:

During this stage the technical team determines the project requirements and how they will be implemented. This entails determining the user interface specifications, system design, and database design. The main results of this stage are the requirements and design specifications of the project.

Development & Test:

The application code, data and another components of the project are built and tested by the technical team. This stage will produce a test plan, code and executable, databases, and other technical aspects of the project.

Deployment:

The system is installed into the clients production envionment, and performance tuning and final testing of the system is executed. This stage will produce the finished software installed into the client's production environment along with an installation and user guide provide for furture reference.

Support:

In this stage the software produced is given to the client along with training on its proper use. In addition, post deployment issues are resloved, and post-deployment feedback is recorded. The main deliverables of this stage are user training and feedback from the client on the sofware produced for the project.
